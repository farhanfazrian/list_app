# Movie List App

### Setup Instalation

- `git clone` repository
-  go to list_app folder projecr
```sh
$ cd list)app
```
 ```sh
$ yarn install
```

- create an .env project and create the API var
```sh
$ yarn start
``` 
REACT_APP_OMDB_API_KEY=YOUR_API_KEY

ex :
REACT_APP_OMDB_API_KEY=1234568
```sh
$ yarn start
```
- open `localhost:3000` in browser

### Tech Stack
- React
- styled-component
